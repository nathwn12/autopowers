Load the `diagnose` skill. Follow the 6-phase process:
1. Build a feedback loop
2. Reproduce
3. Hypothesise (3-5 ranked, falsifiable)
4. Instrument (one variable at a time)
5. Fix + regression test
6. Cleanup + post-mortem

Do NOT propose fixes before completing root cause investigation.
